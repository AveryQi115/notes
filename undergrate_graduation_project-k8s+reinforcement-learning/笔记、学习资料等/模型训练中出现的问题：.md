# 模型训练中出现的问题：

1. 模型只学到不迁移这个决策就停止了

   可能的原因：

   - 数据不均衡：按照现在奖励设置，小于20%或大于80%的少，出现在中间状态的多
   - 模型只学到了位置（像图片里出现一笔的位置，不在乎该位置的深浅），但是想继续学习的时候，轨迹限制了

​		解决办法：

- 改tau(target更新的激进程度)
- 改gamma(未来奖励的占比) 
- 改奖励函数(资源考量从sigmoid->leaky relu)
- replay buffer采样改成顺序(样本均衡采样？)
- 随机动作(如何同时满足可导)
- 模型加dropout
- 改数据集，解决不均衡问题
